# WP Flyout Library

A WordPress library for creating slide-out panels with forms, data displays, and interactive components. Perfect for admin interfaces, edit screens, and anywhere you need contextual editing without page reloads.

## Features

- **Form fields** — text, number, select, AJAX search, toggles, color pickers, image pickers, and more
- **Display components** — headers, stats, timelines, data tables, info grids, price summaries
- **Interactive components** — galleries, file managers, line items, notes, feature lists, key-value pairs
- **Action components** — buttons and dropdown menus with REST API callbacks
- **Stripe-ready** — price config, discount config, payment method display, refund forms
- **Conditional fields** — show/hide fields based on other field values
- **Tabs** — organize fields into tabbed sections
- **Sanitization** — built-in sanitizers for every field type, extensible
- **Hooks** — filters and actions at every stage of the lifecycle
- **WP_List_Table integration** — add flyout actions to any admin list table

## Installation

```bash
composer require arraypress/wp-register-flyouts
```

## Requirements

- PHP 7.4+
- WordPress 5.8+
