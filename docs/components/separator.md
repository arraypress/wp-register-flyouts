# Separator

Visual divider between fields.

```php
'divider' => [
    'type'   => 'separator',
    'text'   => 'Advanced Options',          // Optional label
    'icon'   => 'admin-settings',            // Optional dashicon name
    'style'  => 'line',                      // line, dotted, dashed, double
    'align'  => 'center',                    // left, center, right
    'margin' => '20px',
],
```
